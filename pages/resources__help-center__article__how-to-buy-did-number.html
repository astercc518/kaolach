In this article, we’ll guide you through the process of ordering a DID number on IllyVoIP, setting it up to receive calls on your SIP account, forwarding calls to an international phone number, or routing them to an external SIP server using IP authentication.

What is DID (Direct Inward Dialing)?

DID, or Direct Inward Dialing, is a telephony service that allows businesses and individuals to have virtual phone numbers that route calls directly to specific destinations—such as a SIP account, an external phone number, or an IP-based system.

Instead of requiring a separate physical phone line for each number, a DID number works over VoIP or PBX systems, making it a cost-effective and flexible solution for managing inbound calls.

With DID numbers, businesses can:
Assign unique phone numbers to employees, departments, or services without needing multiple landlines.
Route calls efficiently to a SIP account, mobile, landline, or an external SIP server.
Expand their global presence by having local numbers in different countries, allowing customers to call at local rates.

In short, DID numbers simplify communication, reduce costs, and improve call management—making them an essential tool for VoIP users and businesses worldwide.

To order and use a DID number on IllyVoIP, you first need to have an IllyVoIP account .

Step 1: Log into Your Account

If you already have an account, log in and go to the Phone Numbers section.

Step 2: Choose a DID Number

  • Select the country where you need a number.
  • Check the pricing, setup fees, and any verification requirements needed to obtain the number.

Important: Pricing and verification requirements vary based on country regulations and telecom providers.
Make sure to review them before placing your order.

Please read the Important Notes before placing your order

DID Number Activation and Call Routing Options

Step 1: Configuring DID number

Once your number is active, you can start using it. However, before receiving calls, you need to configure the DID number and choose where to route inbound calls.

Step 2: Choosing Your Call Routing Option

IllyVoIP offers three options for routing inbound calls from your DID number:

  1. Route to SIP Extension

    This option forwards all inbound calls directly to your SIP account, allowing you to answer calls using a softphone or IP phone.

  2. Route to an International Number

    This option forwards calls to a international phone number. For example, if you enter your local SIM card number, any call to your US DID number will be forwarded to your local mobile or landline.

  3. Route to an External SIP Server

    You can route calls to a third-party SIP server, such as 3CX, MagnusBilling, or other softswitches. This is useful if you want to manage calls through an external PBX system.

For more details, check these guides:

  • How to Set UP DID Numbers on 3cx
  • How to Set Up DID Numbers on Magnus Billing

Receiving Calls on Your SIP Account Using IllyVoIP Web Dialer

To receive calls on your SIP account with IllyVoIP, you need to configure your DID number to route calls correctly.

Step 1: Configuring Your DID Number

To set up your DID number to receive calls on your SIP account:

  1. Navigate to DID Configurations and select Route to SIP Extensions.

  2. Review the routing options:

    • Main SIP Account – Your primary SIP account.
    • Agent SIP Account – A different SIP account for agents (Check SIP Accounts section).
    • If you need to create an agent account, follow this guide: [Link]

  3. Select your preferred SIP account.

  4. Save the configuration.

Step 2: Receiving Calls via Web Dialer

If you are logged into your IllyVoIP account and using the web dialer, you should now be able to receive calls through the webphone.

Step 3: Using a Softphone for Incoming Calls (Optional)

You can also configure your SIP account on softphone applications like:

  • Zoiper 5
  • Zoiper 3
  • MicroSIP

For a step-by-step guide on setting up your SIP account with a softphone, check this article: How to Configure SIP on Softphones

Once your setup is complete, your DID number will be fully functional, and you’ll be able to receive calls seamlessly
